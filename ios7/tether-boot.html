Dual Booting 64 Bit devices

Tether boot


It's finally time to boot iOS 7:

Put the device in DFU mode

iPwnder32 -p

Send iBSS (twice) and iBEC

irecovery -f iBSS.img4

irecovery -f iBSS.img4

irecovery -f iBEC.img4

Send and load DeviceTree

irecovery -f devicetree.img4

irecovery -c devicetree

Send Kernelcache and bootx

irecovery -f kernelcache.img4

irecovery -c bootx

We are done! If all goes well the device should boot to the setup screen! Enjoy iOS 7!
